## Домашнє завдання 7 — Додаткові вбудовані SQL функції. Робота з часом

### Завдання

1. Витягти рік, місяць і число з атрибута `date` таблиці `orders`
2. Додати один день до атрибута `date`
3. Відобразити кількість секунд з початку відліку (UNIX timestamp)
4. Порахувати рядки з `date` між `1996-07-10` та `1996-10-08`
5. Вивести JSON-об'єкт `{"id": ..., "date": ...}` для кожного рядка
